6: A group is a collection of users with same permission

    Primary group and secondary group:
        -1: 1 primary group
        -2: 15 secondary groups
        -3: Every new file is owned by User's Primary group
        -4: Primary Groups: Anyone who is apart of the group has access to the file 
        -5: secondary Groups: Users can belong to multiple secondary groups

7: addgroup python
    --> create a new group

8: usermod -a -G python tim
    Assigning users to groups (a stands for append, G stands for group)

9: groups tim   
    See what a group is the  user signing to (tim belongs to python group)

10: groups
    See the group the current user is signing to
